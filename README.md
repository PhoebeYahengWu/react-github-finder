
# React GitHub Finder

## Description
A simple react application that allows users to search for GitHub profile and repos of other GitHub accounts.

## Table of Contents

* Tech/Framework Used

* Usage

## Tech/Framework Used
React, Bootstrap, API


## Usage
Enter the username of the other GitHub account in the search box and press enter key.
